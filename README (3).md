
# Smart Expense Tracker Application

## Overview
This project is a Python-based **Smart Expense Tracker Application** designed to help users log, analyze, clean, visualize, and explore expense datasets.  
It includes concepts of **Control Structures, Arrays, OOP, NumPy, Pandas, Matplotlib, and Seaborn**.

The program is fully menu-driven and suitable for academic and real-world use.
